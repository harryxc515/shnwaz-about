<h2 align="center">📂 PROJECT STRUCTURE</h2>

```
src/
├── 🧩 components/
│   ├── portfolio/         → ProjectCard · ImageLightbox · DetailModal
│   ├── ui/                → shadcn/ui component library
│   ├── CustomCursor.tsx   → Animated cursor with hover states
│   ├── HeroSection.tsx    → Parallax hero banner
│   ├── AboutSection.tsx   → Personal introduction
│   ├── PortfolioSection   → Dynamic project grid
│   ├── MusicSection.tsx   → Music & beats showcase
│   ├── ServicesSection    → Services offered
│   ├── SkillsSection.tsx  → Skills visualization
│   ├── ContactSection     → Contact form
│   ├── Navbar.tsx         → Navigation bar
│   └── Footer.tsx         → Minimal footer
├── 🪝 hooks/              → useParallax · useScrollAnimation · useAuth
├── 📄 pages/              → Index · NotFound
└── 🔌 integrations/       → Backend client & types
```

<h2 align="center">🚀 QUICK START</h2>

```bash
# Clone the repository
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Launch dev server
npm run dev
```
